Css Selectors

Tag Tag Name Description
div .my-class Descendant Selector - will target all elements with a class of "my-class" that are a descendant of a div> element (including a direct child, children of children, etc)
p > .another-class Child Selector - will target any elements with a class of "another-class" that are a direct child of a p> element
#unique-id + div Adjacent Sibling Selector - will target any div> element that is the next sibling of an element with an id of "unique-id"
.class-name ~ p General Sibling Selector - will target any p> elements that are the next siblings of an element with a class of "class-name"
